# 🚀 SEO Content System using AI + Prompt Engineering

## 📌 Overview
This project demonstrates how to build a **real-world SEO content system** using AI and prompt engineering.

Instead of writing random blog posts, this system follows a **content cluster strategy** used by SEO agencies and SaaS companies to improve search rankings and organic traffic.

---

## 🎯 Objective
To design a reusable prompt workflow that can:

- Generate SEO-optimized blog outlines (H1–H3)
- Write long-form pillar content
- Create supporting blog clusters
- Optimize content for local SEO (city + service)
- Align content with business goals (traffic, leads, authority)

---

## 🏪 Business Chosen
**Business Type:** Cafe  
**Location:** Kakinada  
**Target Audience:** Students, working professionals, families, and coffee lovers  

---

## 🧠 SEO Strategy Used

### 🔹 Content Cluster Model
- 1 Pillar Blog (Main Topic)
- 4 Supporting Blogs (Subtopics)
- Internal linking between all blogs
